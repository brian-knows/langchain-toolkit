# Gelato Integration

When creating a `BrianAgent` or when initializing the `BrianToolkit` for using the tools in your own custom agents, you can pass a `gelatoApiKey` option to enable the Gelato integration.

If a `gelatoApiKey` is provided, the Agent (or the tools) will send the transactions using the `GelatoRelay` using the `sponsoredCallERC2771` method.

## Examples

### Using `createBrianAgent`

```typescript
const brianAgent = await createBrianAgent({
  apiKey: BRIAN_API_KEY,
  privateKeyOrAccount: PRIVATE_KEY as `0x${string}`,
  llm: new ChatOpenAI(),
  options: {
    gelatoApiKey: process.env.GELATO_API_KEY,
  },
});
```

### Using the `BrianToolkit`

```typescript
const { tools } = new BrianToolkit({
  apiKey: BRIAN_API_KEY,
  privateKeyOrAccount: PRIVATE_KEY as `0x${string}`,
  options: { gelatoApiKey: process.env.GELATO_API_KEY },
});
```
